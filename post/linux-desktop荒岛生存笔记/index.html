Linux Desktop荒岛生存笔记

 2021-04-26 1268 字 Linux 

第一话

咳咳……

算来将 Linux-desktop 作为主系统已是三载有余了。自三年前，我的表姐帮忙在我那破费了 550 大洋于中关村奸商市场讨来的 ThinkPad x200 装上了 Ubuntu18.04 后，就注定了往后几十年的折腾生涯……

🐶🐶🐶

从 Ubuntu 到 Kali，从 Deepin 到 Manjaro，从 Arch Linux 到 Kubuntu。三年间电脑换了四台，随之而来的便是在物理机上对各发行版的不断尝鲜。兜兜转转一圈过后，最终还是停留在了 Arch Linux + KDE 的组合上 —— 也就是眼下我手中的联想小新 Air14。

如今基本算是能安稳地脱离 Windows（除了部分游戏需求😹），将学习与生活 all in Linux 了。菜狗如我，摸索熟悉了几年，才较为舒适地在 Linux 下生存下来并达到了小康水平。

后话

这里记录一些我个人喜好下的软件解决方案。

IM

腾讯讯，恶心心。

自己不开发 Linux 下的 IM 客户端就算了，还主动阻碍社区自己为爱发电。不仅逐渐停止了 web-qq、web-wechat 的支持，还屡屡干扰 wine-tim、wine-wechat 的可用性。

上面来文件了，说要发展国产 OS 生态。TX 表示双脚赞成，积极参与 —— 于是便整上了 80% segmentation fault 率的 QQ 2008 for Linux 和 套皮糊弄的 Wechat for Linux。

至此已经对腾讯不报任何希望了。结合近期关于微信的科技垄断的负面新闻，已然放弃在 Linux 上折腾任何腾讯的产品。IM for Linux 只得另谋它路。

自己的选择是打着加密通讯、反审查，且拥有5亿多用户的 Telegram。其开源的客户端实现让其能在任何平台上无痛使用，功能性上也是精简而全面。所有必要功能都具有，也未见花里胡哨的冗余功能。

代理

这种东西不做成跨平台和跨架构的软件那才奇了怪了。我用的是 v2ray 的解决方案，外加 Qv2ray 跨平台客户端。（话说传授翻墙知识好像触犯刑法来着……）

音乐客户端

鹅家的 QQ 音乐 for Linux 依然是套皮糊弄……相反网易云音乐 for Linux 的体验良好 —— 就决定是你了 我也没得选。

网盘

网盘这种东西当然是用 GitHub 私有仓库啦

百度网盘 for Linux 体验不错。

即使这次鹅厂的微云依然没有 Linux 客户端，但 Web 端总归是极香的，百度网盘惜败于此。这次我投靠了微云~~（才不是因为 10元/月 的便宜价格呢）~~

阅读

虽然鹅厂丝毫不愿意为 Linux-desktop 写一行代码，但微信读书又赢在了优秀的 Web 端上。

游戏

steam 上也是有上千个支持 Linux 的游戏的说。外加基于 java 的游戏（Minecraft）与 wine 的帮助，凑合着玩？话说 steam 好像实现了效率不错的模拟方案用于在 Linux 下运行 PE 构建的游戏，我也没有太了解……

但 OW、GTA5 等还是让我放弃了格式化 Windows 分区中的第二系统。

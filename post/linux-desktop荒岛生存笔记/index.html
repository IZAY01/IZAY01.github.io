Linux Desktop荒岛生存笔记

 2021-04-26 291 字 Linux 

第一话

咳咳……

算来将 Linux-desktop 作为主系统已是三载有余了。自三年前，我的表姐帮忙在我那破费了 550 大洋于中关村奸商市场讨来的 ThinkPad x200 装上了 Ubuntu18.04 后，就注定了往后几十年的折腾生涯……

🐶🐶🐶

从 Ubuntu 到 Kali，从 Deepin 到 Manjaro，从 Arch Linux 到 Kubuntu。三年间电脑换了四台，随之而来的便是在物理机上对各发行版的不断尝鲜。兜兜转转一圈过后，最终还是停留在了 Arch Linux + KDE 的组合上 —— 也就是眼下我手中的联想小新 Air14。

如今基本算是能安稳地脱离 Windows（除了部分游戏需求😹），将学习与生活 all in Linux 了。菜狗如我，摸索熟悉了几年，才较为舒适地在 Linux 下生存下来并达到了小康水平。
